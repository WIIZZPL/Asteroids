Uaktualnia stan gracza.

Przesuwa go wedle prędkości i przyspieszenia. Jeśli keyboardState wykazuje wciśnięcie klawiszy góra-dół, odpowiednio przyspiesza w tym kierunku; Jeśli natomiast wykazuje lewo-prawo, obraca graczem; A jeśli spację, tworzy nowy obiekt Bullet i umieszcza wskaźnik w vector'ze bullets.
